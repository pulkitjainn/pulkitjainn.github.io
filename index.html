Projects

  • OneStop App | 100+ downloads

    This is a multi purpose utility that contains common & frequently used utility tools (like Stopwatch, Todo List, Calculator, Flashlight, Compass, etc) altogether coded in it. The main aim behind the project was to ease the user experience and eradicate the headache of downloading and maintaining different apps resulting in more utilization of phone memory.

  • Entertainment Adda | 100+ downloads

    This application is a collection of Movies and TV shows. It fetches data from TMDb. The application is the perfect destination for all entertainment freaks. The application contains all the information of the movie or show like genre, cast, synopsis, trailer and also provides features like sharing movies on social media, etc.

  • Hand Recognition

    The program recognises hand from a input image using Single Shot MultiBox Detector-TensorFlow. A huge dataset of hand pictures are trained on SSD-TensorFlow for detecting a hand in almost any of its position from the input image.

  • Color Palette Generator

    The program generates a color palette which shows the frequently occurring colors in the given images using K-Means Clustering. In this the similar color pixels are clustered together and clusters with maximum pixel count represents the frequently occurring colors.

  • Smart Evaluation Tool | Minor Project | Ongoing

    This tool provides the reaction of the user while the user is checking out their products. These reactions can be then processed by the company the way they want in order to compute the rating for the product. This tool eradicates the headache of asking the user to manually enter the rating every time.

  • VoiceCommand | Ongoing

    The main aim behind the application was to provide a completely hands free experience to the user where most of the work can be done by command of voice. Currently the application is only capable of opening all the applications installed on the device. Features like making calls, sending text messages will be incorporated in the future.
